# PC Store

## Présentation
Ce projet est un site e-commerce qui permet de vendre des ordinateurs et des accessoires comme les souris.

## Technologies utilisées
HTML
CSS
PHP
MySQL

## Installation
1 Télécharger le projet
2 Importer la base de données pc_store.sql dans phpMyAdmin
3 Lancer le serveur local avec XAMPP

## Configuration
Modifier le fichier config.php avec les informations de la base de données.

Host : localhost
User : admin
Password :admin
Database : pc_store
